# SkySweep32 — Модульный гайд по апгрейду

## 📦 Аппаратные конфигурации

SkySweep32 спроектирован для **бюджетности и модульности**. Начните с базовой конфигурации и расширяйте по мере необходимости.

### 🟢 Уровень 1: «Starter» (~$15-20 / ~1500₽)

**Что вы получаете**: Базовый мониторинг 2.4 ГГц с веб-дашбордом

| Компонент | Модель | Цена | Назначение |
|-----------|--------|------|------------|
| Микроконтроллер | ESP32 DevKit V1 | ~500₽ | Процессор + WiFi + BLE |
| Дисплей | OLED 128x64 (SSD1306) I2C | ~250₽ | Визуальная индикация |
| РЧ-модуль | NRF24L01+ | ~150₽ | Мониторинг 2.4 ГГц |
| **Итого** | | **~1500₽** | |

**Возможности**:
- ✅ Мониторинг 2.4 ГГц (WiFi, DJI, ExpressLRS)
- ✅ Детекция Remote ID через BLE (бесплатно — встроенный Bluetooth ESP32)
- ✅ Веб-дашборд через WiFi (бесплатно — встроенный WiFi ESP32)
- ✅ OLED дисплей с RSSI в реальном времени
- ✅ 5-уровневая оценка угроз
- ❌ Нет покрытия 900 МГц / 5.8 ГГц
- ❌ Нет GPS / логирования / mesh-сети

**Подключение**:
```
ESP32 → NRF24L01+:
  GPIO 23 (MOSI) → MOSI
  GPIO 19 (MISO) → MISO
  GPIO 18 (SCK)  → SCK
  GPIO 15 (CS)   → CSN
  GPIO 2  (CE)   → CE
  3.3V           → VCC
  GND            → GND

ESP32 → OLED:
  GPIO 21 (SDA)  → SDA
  GPIO 22 (SCL)  → SCL
  3.3V           → VCC
  GND            → GND
```

### 🟣 Уровень 4: «Juggernaut» (~$100-200 / ~10000-20000₽)

**Апгрейд Sentinel**: Архитектура активного подавления (РЭБ)

| Дополнительный компонент | Модель | Назначение |
|--------------------------|--------|------------|
| VCO Генератор (5.8 ГГц) | 5.7-5.9Ghz VCO + RF PA 10W | Подавление Аналог/DJI/Walksnail/OpenIPC |
| VCO Генератор (2.4 ГГц) | 2.4-2.5Ghz VCO + RF PA 10W | Подавление ELRS/DJI/Wi-Fi |
| VCO Генератор (900 МГц) | 868-915Mhz VCO + RF PA 10W | Подавление ELRS/Crossfire |
| VCO Генератор (1.5 ГГц) | 1.5Ghz VCO + RF PA 2W | GPS/ГЛОНАСС Джамминг |
| Транзисторный модуль | MOSFET 4-Channel | Коммутация питания VCO |

**Новые возможности**:
- ✅ 'Умный' купол 5.8 ГГц (рушит цифровые ACK пакеты Walksnail/DJI и слепит Аналог)
- ✅ Широкополосное подавление управления ELRS (900МГц / 2.4ГГц)
- ✅ Автоматическое подавление навигации (GPS L1) при обнаружении любых дронов
- ✅ Управление профилем шума (Sweep) через ЦАП (DAC) встроенный в ESP32

> **ВНИМАНИЕ:** Для работы DAC используются пины **GPIO 25 и 26**. Если вы используете LoRa, перенесите её пины (RESET и CS) на GPIO 12 и 14.
